# Reporting Tools Platform Site

A lightweight GitHub Pages support portal for Reporting Tools Platform.

This site provides simple, crawlable navigation to the main platform, its reporting tools and its educational guides.

## Main Website

https://reporting-tools-platform.vercel.app/

## GitHub Pages Portal

https://kike-pinto.github.io/reporting-tools-platform-site/

## Purpose

The support site was created to:

- Provide an additional public entry point to Reporting Tools Platform
- Link directly to report generators
- Organize links to reporting guides
- Improve navigation between project resources
- Support project discovery through GitHub Pages
- Present the platform outside the main Next.js application

The main product and all reporting functionality remain hosted on Vercel.
